Methods

(static) throwNotFoundResponse()

Lève une réponse HTTP indiquant que la page demandée n'a pas été trouvée. Cette fonction lance une instance de Response avec le message "Page Not Found", le status 404 et le statusText "Not Found". Elle est utilisée notamment dans les loaders pour signaler qu'aucune ressource correspondante n'a été trouvée.
