Module: RadarPerformanceChart

Composant de graphique radar pour afficher la performance de l'utilisateur. Ce composant utilise la bibliothèque Recharts pour rendre un graphique radar qui compare différentes mesures de performance (intensité, vitesse, force, endurance, énergie, cardio). Les données sont triées selon un ordre défini et étiquetées en français pour l'affichage.
